What Counts Here

Official Monday miles come from documented Morning Run Club runs. For each run, the tracker uses the distance and the number of runners who were there.

Official Monday Miles = Distance × Runners Present

Some older runs were posted before we started tracking distance and runners, so this page only counts runs that actually have that data.

Total member mileage would be different. That would include miles runners complete outside Monday meetings, like regular training runs. That part is planned for later.

Strava Integration Plan

The long-term goal is to let runners connect Strava once and have their runs automatically count toward the club mileage total. That feature is not active yet because Strava API access takes extra setup.
