Running with Purpose

Morning Run Club is a student-led club built around a simple idea: start the week with a Bible verse and a run. Every Monday morning, members meet before school to run together, build discipline, and start the week with purpose.

Verse of the Week

Before each run, someone shares a Bible verse. The goal is not to make the club complicated. It is to take a few minutes to focus on faith before running and going into the school day.

Our Core Values

  • Faith – We start with Scripture and remember that our strength comes from God.
  • Discipline – We show up even when it is early, cold, or inconvenient.
  • Community – We run together and encourage each other.
  • Service – We use running to support causes beyond ourselves through charity race efforts.
