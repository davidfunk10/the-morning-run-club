Source code for the Morning Run Club website.

The Morning Run Club is a student-led club at Christ Church Episcopal School. Its mission is to promote purposeful running. Monday morning runs are held weekly, where members share the verse of the week and go for a run together. The club values faith, physical fitness, and the joy of running. It also hosts charity races that raise money for local charities such as the Ronald McDonald House.
